# EngageChain

EngageChain is an Intelligent Contract built on GenLayer that converts subjective opinions into verifiable on-chain verdicts through AI and decentralized consensus.

---

## Protocol Flow

✍ Human Input → ◉ AI Evaluation → ⬡ Optimistic Democracy → ⛓ On-Chain Record  

- **Human Input** — the user submits an opinion, proposal, or dispute  
- **AI Evaluation** — the contract calls the LLM via `gl.exec_prompt()` to analyze the text  
- **Optimistic Democracy** — GenLayer validators execute the same prompt and compare results using the Equivalence Principle  
- **On-Chain Record** — the consensus verdict is stored permanently and immutably

## Contract: `contract/engagechain.py`

Written in Python using the GenLayer SDK.

### Persistent State

| Field | Type | Description |
|------|------|------------|
| submissions | TreeMap[u256, str] | Original text of each opinion |
| ai_responses | TreeMap[u256, str] | JSON response generated by AI |
| verdicts | TreeMap[u256, str] | Final consensus verdict |
| authors | TreeMap[u256, str] | Author address |
| statuses | TreeMap[u256, str] | pending / evaluated / finalized |
| timestamps | TreeMap[u256, u256] | Logical block of the entry |
| total_submissions | u256 | Global entry counter |

---

## Public Functions

| Function | Type | Description |
|---------|------|-------------|
| submit_opinion(text) | write | Registers a new opinion, returns ID |
| evaluate_opinion(id) | write | Calls the LLM for evaluation (non-deterministic) |
| store_ai_response(id, response) | write | Stores external AI response (author only) |
| finalize_opinion(id, verdict) | write | Finalizes with consensus verdict |
| get_opinion(id) | view | Reads original text |
| get_ai_response(id) | view | Reads AI response |
| get_verdict(id) | view | Reads final verdict |
| get_status(id) | view | Reads status (pending/evaluated/finalized) |
| get_author(id) | view | Reads author address |
| get_total_submissions() | view | Reads total number of opinions |
| get_full_entry(id) | view | Returns all data as a JSON string |

## Deployment

### Option 1 — GenLayer Studio (recommended for testnet)

1. Go to GenLayer Studio  
2. Create a new contract and paste the contents of `contract/engagechain.py`  
3. Deploy it and copy the contract address  
4. Paste that address into **Network Settings** in the frontend  

---

### Option 2 — Local CLI

```bash
# Install GenLayer CLI
pip install genlayer

# Deploy contract
genlayer deploy contract/engagechain.py

# Interact
genlayer call <contract_address> submit_opinion "My sample proposal"
genlayer call <contract_address> evaluate_opinion 0
genlayer call <contract_address> get_full_entry 0
